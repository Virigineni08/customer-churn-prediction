# Customer Churn Prediction 📊

## Overview
A Machine Learning model to predict whether
a telecom customer will churn or not.

## Dataset
Telco Customer Churn Dataset
- 7043 records
- 21 features

## Model Used
Random Forest Classifier

## Accuracy
~79%

## Libraries Used
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## Steps
1. Data Cleaning
2. Exploratory Data Analysis
3. Feature Encoding
4. Model Training
5. Model Evaluation
6. Prediction
